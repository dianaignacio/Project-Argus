To install Reactive Extensions - System.Reactive Library, run the following command in the Package Manager Console
	PM> Install-Package Rx-Main -Version 1.0.2856

To install protobuf-net, run the following command in the Package Manager Console
	Install-Package protobuf-net

To install IronPython, run the following command in the Package Manager Console
	Install-Package IronPython

To install log4net, run the following command in the Package Manager Console
	Install-Package log4net
